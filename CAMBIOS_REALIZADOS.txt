INSTRUCCIONES:
──────────────

1. Abre el archivo: config.js (está en la raíz del proyecto)

2. Busca estas líneas:
   
   window.AppConfig = {
       supabaseUrl: "TU_SUPABASE_URL",
       supabaseAnonKey: "TU_SUPABASE_ANON_KEY"
   };

3. Reemplaza:
   - TU_SUPABASE_URL → Tu URL real de Supabase
   - TU_SUPABASE_ANON_KEY → Tu clave anónima real

4. Guarda el archivo

5. ¡LISTO! Se cargan automáticamente en TODAS las páginas

¿PREGUNTAS O PROBLEMAS?
════════════════════════════════════════════════════════════════════

1. Asegúrate de haber reemplazado las credenciales de Supabase
2. Abre la consola del navegador (F12) y busca errores
3. Verifica que localStorage esté habilitado
4. Comprueba que Supabase tenga las tablas correctas:
   - profiles (id, username, level, exp, ...)
   - item_states (user_id, category, item_id, fav, viewed, ...)
